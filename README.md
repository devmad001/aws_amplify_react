## Setup
```
yarn
``` 

## Running the Project
```
yarn start
``` 

## Build & Deploy
### Staging
```
yarn dist:staging 
``` 
```
yarn deploy:staging 
``` 

### Production
```
yarn dist:prod 
``` 
```
yarn deploy:prod 
``` 

## Built With
- Firebase (Authentication + Hosting)
- Hasura (Data + Authorisation)
  - Hasura Cloud
- Apollo GraphQL
- React Admin
- Webpack
